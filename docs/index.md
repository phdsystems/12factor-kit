# 12-Factor Reviewer Documentation

**Audience**: All users, developers, and contributors

## WHAT

12-Factor Reviewer is a language-agnostic CLI tool for assessing 12-Factor App compliance. This hub provides navigation to all project documentation.

## WHY

Centralized documentation access prevents fragmentation and ensures all audiences — users, developers, and architects — can find what they need.
